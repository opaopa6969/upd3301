upd3301 / PC-8801 emulator

NEC μPD3301 CRTC を起点に、PC-8001 / PC-8801 をまるごとブラウザで動かす純JavaScriptエミュレータ。依存ゼロ・決定論・ヘッドレステスト可能。Z80×2、μPD765 FDD、YM2203/YM2608(OPNA) まで。

▶ PC-8801 を起動する μPD3301 テキストデモ GitHub

ROMは同梱していません（後述のBYO-ROM）。ブラウザ内だけで動き、データはどこにも送信されません。

何ができる

  • PC-8801 (N88-BASIC) をディスクから起動（FDDサブCPU＝2個目のZ80まで実装）
  • PC-8001 (N-BASIC) 起動、μPD3301＋μPD8257 DMA の実挙動
  • サウンド: YM2203(OPN) FM3+SSG3、サウンドボードII (YM2608/OPNA) FM6＋リズム
  • クリーン表示 / CRTシミュレーション（蛍光体残光・管面・走査線）切替
  • 速度コントロール ×0.25〜×16（早送り／スロー）
  • 巻き戻し（決定論スナップショットで時間を遡る）
  • WebM録画（画面＋音声）
  • 保存済みディスク選択（IndexedDBに一度入れれば次回から自動起動）
  • ICE / IDE デバッガ、逆アセンブラ、Z80アセンブラ内蔵
  • 決定論: 同seed＋同入力→同出力。`node --test` で全数検証

使い方（BYO-ROM）

著作権上、NECのROMは同梱できません。実機から吸い出した自分のROMを用意してください。PC-8801デモで:

  1. 「PC-8801 BIOS一式(n88)」ボタンで n88.rom + n88_0..3.rom + disk.rom（計6ファイル）をまとめて選ぶ → 自動でN88モードへ
  2. ディスク(.d88) を選ぶ（ドライブ0にマウント）
  3. リセット → ブータブルなゲームディスクなら起動

一度読み込めばブラウザのIndexedDBに保存され、次回から自動起動します。ローカルに roms/<機種>/ を置いて manifest.json に "n88": "<dir>" を書けば、サーバから自動ロードもできます。

技術メモ

  • 純JS・依存ゼロ: レジスタin → サンプル/ピクセルout
  • 決定論: Math.random不使用、状態はすべてスナップショット可能
  • 2個のZ80: 本体CPU＋FDDサブCPUが8255ペアでハンドシェイク
  • OPN/OPNA: LOG-SIN→EXP の本物の信号経路、ボード出力段まで再現
  • ヘッドレステスト: node --test（250+テスト）
