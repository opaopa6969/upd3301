# upd3301 — PC-8801 I/Oポートマップと実装状況

`machine88.js`（`in()` / `out()`）に実装した PC-8801 I/Oポートの作業用リファレンス。
**実装から逆算して**書いてある ── 各ポートが何をするか、うちがモデル化してるか、して
ないなら**なぜそれで安全か**を記録する。ポート集合と挙動は **M88**（`bubio/M88M`移植）
と突き合わせ済み。[`../m88ref/`](../m88ref/) のヘッドレス参照機で、うちの挙動をゲーム
ごとに diff した。だからここの「未実装」は「不明」ではなく「監査した上で意図的に省いた」
という意味。

凡例: **✔ 実装** · **≈ latch のみ**（値は保持、モデル化した範囲に副作用なし）· **○
視覚/opt-in**（コアは捕捉、描画はブラウザ側で未検証）· **✗ 非モデル化**（理由付き）。

## 注目機能

### テキストウィンドウ（port 70h / 78h）── 事件の犯人
`0x8000–0x83FF` の1KB領域が、`(port31 & 6) == 0` かつ非N80モードのとき
`ram[txtwnd + (addr & 0x3FF)]` にマップされる（`txtwnd = port70h << 8`、`OUT 78h`
で1ページ進む）。`IN 70h` は `txtwnd >> 8` を返す。

N88-DISK-BASICのプログラムローダは、載せたプログラムを `0x0001+` に格納し、**この窓
越しに読み返して**（`OUT 70h,00` で `0x8001` が `0x0001` をエイリアス）「まだ読むもの
があるか」を判定する。窓が無いと読み返し=0 → ローダが早期に止まる。これが**軽井沢
誘拐案内**（と他数本）の起動を止めていた唯一の欠落機能。捜査記録参照。M88
`Memory::Out70` / `Update80` に準拠。

### PCG — Programmable Character Generator（port 00h–02h）
`02h/01h` で14ビット番地、`00h` でデータバイトを作り、番地ビット12を立てた書き込みが
字形1走査線を `pcgRam[addr & 0x3FF]` に格納する（字形 `0x80–0xFF`、各8行）。捕捉は
コア側で検証可能。視覚オーバーレイは `renderScreen({ pcg })` の opt-in で、ブラウザ側
のため現状**未検証**。PC-8001の PCGアドオン（PCG8100）も同じポートを使う。

### μPD1990AC カレンダ時計（port 10h + 40h）
シリアルRTC。`OUT 10h` が3ビットコマンド + シリアルinビットを読み込み、`OUT 40h` が
エッジでクロックする（`b1` CSTB=コマンドラッチ、`b2` CCLK=シフト）、`IN 40h b4` が
シリアルout。コマンド3で日付をシフトレジスタへ、コマンド1でLSBから読み出す。返す日付
はスナップショット/リプレイ再現性のため**固定の決定論デフォルト**。ライブ時計は
`rtcDate` コンストラクタ引数で注入可能。M88 `Calender` に忠実。

## 意図的に非実装

- **40h GVRAMウェイト(b4)** ── M88はメモリウェイトプロファイル選択に使うが、うちの
  タイミングはページ毎ウェイトを持たない近似なので無効。全ゲーム起動に支障なし。
- **99h CD-BIOS, F0h/F1h 辞書ROM, E2h/E3h EMM** ── 非搭載のROM/RAMのバンク/窓選択。
  読み返し用にlatchするのみで副作用なし（該当オプション非搭載時のM88と一致）。
- **FGU（Functional Graphics Unit）** ── PC-8001用の外付け640×200グラフィックボード
  （FGU-8000/8200）。PC-8801は640×200 GVRAMを内蔵（うちも実装済み）なので、FGUは
  それを狙い撃つPC-8001モードsoftでのみ意味を持つ。非モデル化。

## 監査手法

ポート一覧は、うちの `in()`/`out()` を M88 の IOBus コネクタ表（`src/pc88/pc88.cpp`）
と diff して得た。次に両エミュを同一ディスクでヘッドレス実行し（`m88ref/`）、ゲームが
実際に叩く全ポートが「実装済み」か「証明可能に無害」かを確認した。参照神託の手法と、
それを再現する agent用prompt は [`../m88ref/README.ja.md`](../m88ref/README.ja.md)。
